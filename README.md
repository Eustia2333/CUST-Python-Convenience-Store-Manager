**优选便利店智能管家系统**

长春理工大学数据科学与大数据技术专业 大二上学期 Python程序设计为期两周的课程设计作业，没什么含金量，主要为了学习一下GitHub的操作流程，所以上传了。

**环境**

Python 3.11, MySQL, Tkinter, Matplotlib.

**功能**

两个账号权限，分别是店长和收银员。店长可以对商品增删查改，可以看到经营报表和流水，可以管理员工账号。商品有临期和补货提醒；

收银员的收银系统可以修改订单，生成小票，有会员积分系统。

**结构**

src/: 源代码

docs/: 项目文档

**备注**

预设数据直接看db_setup.py就可以，运行db_setup.py可以直接把预设数据注入（可选）。

运行main.py可以直接运行程序。
